## Core Features

### [Export & Share](export.md)

Export your diagrams in multiple formats:

- **PNG** - High-resolution images for presentations
- **SVG** - Scalable vector graphics
- **Visio (VSDX)** - Microsoft Visio format
- **draw.io** - Draw.io XML format
- **PDF** - Print-ready documents

Share diagrams via links, embed in documentation, or download for offline use.

**[Learn About Export →](export.md)**

### [Collaboration](collaboration.md)

- Real-time collaboration
- Comments and annotations

- Activity tracking

**[Start Collaborating →](collaboration.md)**

### [Web IaC Editor](web-editor.md)

Edit Infrastructure as Code directly in your browser:

- Syntax highlighting

- Live diagram updates
