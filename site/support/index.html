    Before Contacting Support#

    To help us help you faster:

    1. Check Documentation - Your answer might be in the docs
    2. Search FAQ - Common issues are covered
    3. Check Community - Others may have asked the same question
    4. Gather Information:
      • Error messages
      • Steps to reproduce
      • Screenshots
      • Browser/OS information

    Bug Reports#

    Found a bug?

    1. Open an issue
    2. Use the bug report template
    3. Include:
      • Steps to reproduce
      • Expected behavior
      • Actual behavior
      • Screenshots/logs
      • Environment details

    Feature Requests#

    Have an idea?

    1. Start a discussion
    2. Describe the feature
    3. Explain the use case
    4. Get community feedback

    Security Issues#

    Important: For security vulnerabilities, email support@ganak.ai directly. Do not post publicly.

    We follow responsible disclosure and will:

    • Acknowledge receipt within 24 hours
    • Investigate promptly
    • Provide updates on progress
    • Credit you (if desired) when fixed
